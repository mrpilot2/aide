std::shared_ptr< SettingsInterface > AideSettingsProvider::unversionableSettings ( ) const
overridevirtual

use this for settings that should not be stored in a version control system. If the implementation of SettingsInterface is file based, this creates a separate file for settings to be stored in version control.

The user shall be able to easily transfer application settings onto another computer.

A setting that can be stored in version control is for example the selected application language. A setting that should not be stored in version control is for example the window geometry, as it might change with every application exit.
